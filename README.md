> 适用于 ClassIsland v2

**WeatherIsland** 是一款为 [ClassIsland](https://classisland.tech/) 开发的简易天气插件。它通过**和风天气 API** 获取实时天气信息，并在主界面或组件中展示。

---

## ✨ 功能

- **实时天气显示** – 获取当前温度、天气状况、风力、湿度等关键信息。

## 🔧 配置

### 1. 获取和风天气 API Key

- 前往 [和风天气开发者平台](https://dev.qweather.com/) 注册并创建应用，获取 API Key

### 2. 在 ClassIsland 中配置

启用插件后，进入 `设置 → 插件 → WeatherIsland`，按照引导配置

---

## 🖥️ 使用

插件会在 ClassIsland 主界面添加一个天气组件，展示：

- 当前气温和天气图标
- 简短天气描述（如“晴”、“小雨”）

你也可以在“组件管理”中调整天气组件的位置和大小。
